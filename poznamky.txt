1. Instalacia django do pythonu:
    pip install django
    pip list
2. Vytvorenie ddjango projektu:
    django-admin startproject nazov_projektu .
3. Nastavenie slovenciny pre projekt (mysite/settings.py):
    LANGUAGE_CODE = 'sk'
    TIME_ZONE = 'CEST'
4. Spustenie vyvojove servera:
    python manage.py runserver
    python.exe .\manage.py runserver
5. Vytvorenie aplikacie:
    python manage.py startapp nazov_apliklacie
6. Zaregistrovanie aplikacie do projektu (nazov_projektu/settings.py):
    INSTALLED_APPS = [
        'django.contrib.admin',
        'django.contrib.auth',
        'django.contrib.contenttypes',
        'django.contrib.sessions',
        'django.contrib.messages',
        'django.contrib.staticfiles',
        'nazov_apliklacie',
    ]

10. CSS, Obrazky
- vytvorit v priecinku aplikacie priecinok static/nazov_aplikacie
- v tomto priecinku vytvorit CSS (napr. kalkulacka.css )
- do tohto priecinku nakopirovat obrazky
- do template (index.html) doplnit:
    - uplne na zaciatok {% load static %}
    - do hlavicky odkaz na style
    <head>
        <link rel="stylesheet" href="{%static 'kalkulacka/kalkulacka.css'%}">
    </head>
- obrazok vlozit:
<img src={%static 'kalkulacka/calc.png'%}>
11. Modely (models.py)
class Osoba(models.Model):
    meno = models.CharField(max_length=100)
    priezvisko = models.CharField(max_length=100)
    vek = models.IntegerField()
    datum = models.IntegerField()
    email = models.EmailField()
    muz = models.BooleanField()

    def __str__(self):
        return f"{self.meno} {self.priezvisko}"

12. Vytvorenie struktury databazy podla modelov:
python manage.py makemigrations

databaza sa defaultne vytvara v db.sqlite
